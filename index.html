Широкі аудиторії

Вигорання кластеру комунікації

При широких аудиторіях вигорає не аудиторія, а група людей які реагують на конкретний меседж
Що сталосьЯк виглядає в метрикахЩо це значитьЩо робити
Кластер вичерпаний Hook Rate падає, CPM росте, Spend падає. Алгоритм не може знайти нових людей для цього меседжу. Алгоритм вже показав цей креатив усім, хто б відреагував. Pool вичерпаний. Інший кут для іншого кластеру людей. Не рефреш — а новий меседж.
Кластер втомився Hook Rate тримається, CTR падає. Люди бачать, але вже не клікають. Бачили цей меседж занадто багато разів. Frequency > 3-4. Рефреш хуку: нова перша секунда, інший візуал, той самий меседж.
Кластер не конвертує CTR тримається, але CR впав. Люди клікають, але не купують. Алгоритм розширив покази на менш релевантних людей або лендінг не відповідає. Перевірити message match (креатив ↔ лендінг). Або зменшити бюджет щоб алгоритм звузив.

Алгоритм дій: 5 питань

1. Hook Rate впав > 20%?
→
Так: хук більше не чіпляє. Новий хук або новий кут. Ні: йдемо далі.
2. CPM виріс > 25%?
→
Так: кластер вичерпується. Якщо тільки по одному креативу — замінити. Якщо по всіх — зовнішній фактор, перечекати.
3. Це один креатив чи всі?
→
Один: вигорів креатив (новий кут). Всі по продукту: вигорів оффер (новий оффер). Все скрізь: зовнішній фактор.
4. CTR тримається, CR впав?
→
Так: проблема не в креативі, а в лендінгу або алгоритм розширив аудиторію. Перевірити message match. Зменшити бюджет на 20%.
5. Spend різко впав (>30%)?
→
Так: алгоритм зупинив сам — кластер повністю вичерпаний. Не намагайтеся "реанімувати". Запускайте новий креатив з іншим кутом.

Winner > 30 днів, ще працює планово Почати готувати заміну зараз. Не чекати поки впаде. Мати 2-3 готових кути в резерві. Безшовна заміна без провалу в обороті
Головне правило: не реагуйте на одноденні коливання. Дивіться тренд 3 дні. Якщо метрика падає 3 дні підряд — це сигнал. Один поганий день — це шум.
